# Lab 14 - Integration & API Testing

### Төслийн тухай
Энэхүү бие даалтаар Postman ашиглан API тест хийх, автоматжуулах болон Integration testing-ийн үндсэн ойлголтуудыг практик дээр хэрэгжүүллээ.

### Ашигласан багаж хэрэгсэл
* **Postman**: API хүсэлт болон тест бичих
* **JSONPlaceholder**: Тест хийх mock API
* **Git & GitHub**: Хувилбарын удирдлага болон workflow

### Гүйцэтгэсэн ажлууд
* **Part A**: Environment тохируулж, BaseURL хувьсагч ашиглан GET хүсэлт амжилттай явуулсан.
* **Part B**: Автомат тестүүд (Status code, Response time) бичиж, Request Chaining ашиглан нэг хүсэлтийн хариуг дараагийн хүсэлтэд дамжуулсан.

### Үр дүнгүүд
* Бүх тестүүд амжилттай (Passed).
* API хүсэлтүүд хоорондоо хамааралтайгаар зөв ажиллаж байна.
